Exercises - Methods

  1. Write a method isPrime(int n) that returns true if n is prime.
  2. Create a method greet(String name) that prints a custom message.
  3. Write a method maxOfThree(int a, int b, int c) that returns the largest.
  4. Write a method that calculates the sum of all numbers from 1 to n.
  5. Overload a method area() for square and rectangle.
