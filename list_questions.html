Exercises - List

  1. Create a List<String> with 3 food items. Print each in uppercase.
  2. Write a method that accepts a List<Integer> and returns the product of all numbers.
  3. Create a List<String> and remove all strings shorter than 4 characters.
  4. Sort a list of integers using Collections.sort() and print the result.
  5. Given a List<String>, count how many items contain the letter 'e'.
