Exercises - Mixed

  1. Build a Student class with encapsulated fields. Store students in an ArrayList. Write a method to print students older than 20.
  2. Create an abstract class Appliance with method start(). Implement it in WashingMachine and Oven. Use method overriding.
  3. Create a method that accepts an array of integers and returns a new ArrayList<Integer> with only even numbers.
  4. Design a program with a Vehicle base class and subclasses Car and Bike. Override a method move(), and use an array of Vehicle to store and call their move methods.
  5. Write a program that:
    Accepts 5 names into an array.
    Converts the array to a List.
    Removes all names shorter than 5 characters.
    Prints the list using a method printList(List<String>).
