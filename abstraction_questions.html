Exercises - Abstraction

  1. Create an abstract class Animal with method makeSound(). Implement it in Dog and Cat.
  2. Create an abstract class Employee with calculateSalary(). Extend with Intern and Manager.
  3. Make an interface Playable with play(). Implement in Video and Audio.
  4. Abstract class Shape has area(). Implement it in Circle and Square.
  5. Interface Drivable with start() and stop(). Implement in Bike and Truck.
