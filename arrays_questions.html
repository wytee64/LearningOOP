Exercises - Arrays

  1. Create an array of 5 integers and write a method to return the highest number.
  2. Create a String[] with 4 names. Print all names that start with the letter "A".
  3. Write a method that receives an int[] and returns the average of all numbers.
  4. Create a 2D array of 3 rows and 3 columns. Fill it with values and print the sum of each row.
  5. Given an array of integers, write code to reverse it without using a second array.
