Exercises - Polymorphism

  1. Method Overriding: Create base class Shape with draw(). Override in Circle and Triangle.
  2. Method Overloading: Create print(String) and print(int) in the same class.
  3. Create a Media array with objects of Song, Video, Podcast (all extend Media). Call a common method using loop.
  4. Create a Payment class with method pay(). Override it in CreditCard and Cash.
  5. Write a method add() that works with both int and double using overloading.
