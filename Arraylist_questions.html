Exercises - Arraylist

  1. Create an ArrayList<String> of 5 cities. Remove the last city and print the remaining.
  2. Write a method that takes an ArrayList<Double> of prices and returns the total.
  3. Create an ArrayList<Integer> and remove all odd numbers.
  4. Store 3 student names in an ArrayList, then add 2 more names at index 1 and 2.
  5. Clone an ArrayList<String> and confirm the cloned list is separate from the original.
