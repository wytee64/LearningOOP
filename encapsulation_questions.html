Exercises - Encapsulation

  1. Create class User with private fields username, password. Add getters/setters.
  2. Class Account has private balance. Add methods deposit() and withdraw() with validation.
  3. Make a class Student with private age. Ensure age cannot be set to a negative number.
  4. Add validation inside setters to reject null values for name.
  5. Car class with private speed. Control it using methods accelerate() and brake().
