Exercises - Inheritance

  1. Class Vehicle has method move(). Extend it in Car and Plane, override move().
  2. Create base class Person, subclasses Student and Teacher with additional fields.
  3. Class Device has method powerOn(). Extend it with Phone and Laptop.
  4. Multilevel: LivingBeing → Animal → Bird. Each adds its own method.
  5. Use constructor chaining: Person → Employee → Manager. Initialize fields step-by-step.
