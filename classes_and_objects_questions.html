  1. Create a way to represent a book. Each book has a title, author, and whether it's available or not.

    Create a separate class that represents a library. It should store multiple books.
    The library should have a method to add books and another to list all available books.
    Write code that adds at least 3 books, borrows one, and then prints the available books before and after borrowing.
