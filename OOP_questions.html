Exercises - just OOP

  1. Student Info System
    Create a Student class with name, age, and grade.
    Add methods to display student info and update grade.
    Focus on encapsulation (use private fields + getters/setters).

  2. Animal Sounds
    Create a base class Animal with a method makeSound().
    Subclass it into Dog, Cat, and Cow, each overriding makeSound().
    Create a list of animals and call makeSound() on each.
    Focus on inheritance and polymorphism.

  3. Shape Area Calculator
    Create an abstract class Shape with an abstract method getArea().
    Implement Circle, Rectangle, and Triangle classes that extend Shape.
    Write a method to calculate and print the area of each shape.
    Focus on abstraction and using abstract classes properly.

  4. Employee Salary System
    Create a base class Employee with name and baseSalary.
    Create subclasses FullTimeEmployee and PartTimeEmployee.
    Each subclass should have its own logic for calculating total pay.
    Add a method in each class: calculatePay().
    Create a list of employees and print their total salaries.
    Focus on inheritance, overriding, and runtime polymorphism.

  5. Bank Account System
    Create an abstract BankAccount class with account number and balance.
    Add deposit and withdraw methods with validation.
    Create SavingsAccount and CurrentAccount subclasses with their own withdrawal rules (e.g., min balance or overdraft limit).
    Add a Bank class that manages multiple accounts.
    Implement methods to add new accounts, remove accounts, and find accounts by number.
    Focus on abstraction, inheritance, encapsulation, and managing objects in a collection.

  6. Vehicle Management System
    Create a base class for Vehicle with fields like brand, model, and maxSpeed.
    Add a method startEngine() that prints a generic message.
    Create two subclasses: Car and Motorcycle, each overriding startEngine() with their own version.
    Each subclass should have one additional field unique to it (e.g., Car has number of doors, Motorcycle has helmet storage).
    In your main class, create an array or list of Vehicle references storing both Car and Motorcycle objects.
    Loop through the list and call startEngine() on each object.
